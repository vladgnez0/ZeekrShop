# ZeekrParts

Интернет-магазин автозапчастей для автомобилей Zeekr. Построен на Django 4.2 с шаблонами на Tailwind CSS.

## Возможности

### Каталог
- Категории товаров с иерархией (вложенные категории)
- Бренды производителей
- Карточки товаров с множественными фото, характеристиками (атрибутами) и совместимостью по моделям авто
- Фильтрация по категориям, брендам, наличию и диапазону цен
- Сортировка (популярные, новые, по цене, по названию)
- Полнотекстовый поиск с живыми подсказками (AJAX)
- Пагинация

### Корзина и заказы
- Сессионная корзина (добавление, изменение количества, удаление)
- Оформление заказа с автозаполнением данных из профиля
- Статусы заказов: Новый, Оплачен, Отправлен, Завершён, Отменён
- Платежи (Карта / МИР / СБП) — сохранение метода и статуса

### Личный кабинет
- Регистрация и авторизация
- Профиль пользователя (телефон, дата рождения)
- Управление адресом доставки
- Гараж — привязка своих автомобилей Zeekr (модель, год, VIN)
- Избранное (добавление/удаление товаров)
- История заказов с детализацией
- Смена пароля
- Сброс пароля через email

### Контент
- Страница «О нас» с командой, преимуществами и статистикой (управляется через админку)
- Контакты — форма обратной связи с карточками контактов и встроенной картой
- Статические страницы (произвольный HTML/Markdown контент)
- Настройки сайта (название, телефон, email, адрес, соцсети, карта)

### Админ-панель
- Полноценное управление каталогом: товары, категории, бренды, характеристики
- Inline-редактирование фото товаров, значений атрибутов и совместимости
- Управление заказами, платежами и отзывами
- Модерация сообщений с контактной формы
- Управление контентом страницы «О нас» (преимущества, показатели, команда)

## Установка и запуск

### Требования

- Python 3.8+

### Шаги

1. **Клонируйте репозиторий** и перейдите в директорию проекта:
   ```bash
   cd zeekr
   ```

2. **Создайте виртуальное окружение** и активируйте его:
   ```bash
   python3 -m venv venv
   source venv/bin/activate      # Linux / macOS
   venv\Scripts\activate         # Windows
   ```

3. **Установите зависимости:**
   ```bash
   pip install django pillow
   ```

4. **Примените миграции:**
   ```bash
   python manage.py migrate
   ```

5. **Создайте суперпользователя** (для доступа к админ-панели):
   ```bash
   python manage.py createsuperuser
   ```

6. **Запустите сервер разработки:**
   ```bash
   python manage.py runserver
   ```

7. Откройте в браузере:
   - Сайт: [http://127.0.0.1:8000](http://127.0.0.1:8000)
   - Админ-панель: [http://127.0.0.1:8000/admin/](http://127.0.0.1:8000/admin/)

## Модели данных

- **SiteSetting** — глобальные настройки сайта (контакты, соцсети, карта)
- **StaticPage** — статические страницы с произвольным контентом
- **ContactMessage** — сообщения из формы обратной связи
- **Category** — категории товаров (с поддержкой вложенности)
- **Brand** — бренды / производители
- **Product** — товары (артикул, цена, старая цена, остаток, описание)
- **ProductImage** — фотографии товаров
- **ProductAttribute / ProductAttributeValue** — характеристики товаров
- **CarModel** — модели автомобилей Zeekr
- **ProductCompatibility** — совместимость товара с моделью авто
- **UserCar** — автомобили пользователя (гараж)
- **Order / OrderItem** — заказы и позиции заказов
- **Payment** — платежи
- **Review** — отзывы к товарам
- **AboutPage / AboutFeature / AboutStat / TeamMember** — контент страницы «О нас»
- **UserProfile** — расширенный профиль пользователя
- **UserAddress** — адрес доставки
- **Favorite** — избранные товары
